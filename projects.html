Systems Engineering & Analysis project

Dit project was zeer interessant door de vele nieuwe dingen die we erdoor geleerd hebben zoals user stories opmaken en allemaal ander voorbereidend werk. Ik heb dit project gedaan met 3 andere studenten. We hebben een web applicatie gemaakt voor een fictief bedrijf. De bedoeling was om een web applicatie te maken waarop een quiz getoond, gemaakt en aangepast kan worden. Maar we hadden ook een antwoord doos waarop de quizers hun antwoord konden doorsturen.

API project

Dit project was leuk en ben ik trots op omdat ik merkte dat het echt gewoon heel vlot ging ik kon code schrijven alsof het niets was en moest er bijna niet bij nadenken. De bedoeling was om een API te maken die alle data van een zoo kon opslagen en dynamisch gebruikbaar maken. Ook heb ik een andere API geconsumed in de bijna om bij elke cat die aangemaakt werd een kat-feitje toe te voegen.
