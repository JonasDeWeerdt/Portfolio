Project: Web Applicatie Quiz

Context en achtergrond: Dit project was zeer interessant door de vele nieuwe dingen die we erdoor geleerd hebben, zoals het opmaken van user stories en andere voorbereidende werkzaamheden. We hebben een web applicatie gemaakt voor een fictief bedrijf waarop een quiz getoond, gemaakt en aangepast kan worden. Daarnaast was er ook een antwoord doos waarop de deelnemers hun antwoord konden doorsturen.
Wat heb ik concreet gerealiseerd: Een API om users mee aan te maken en de volledige quiz te controleren.
Mijn aandeel in groepswerk: Ik heb dit project samen met 3 andere studenten gedaan. Ik heb vooral aan de API gewerkt en geholpen met de frontend.
Wat ik heb geleerd: Tijdens dit project heb ik geleerd hoe ik efficiënt kan samenwerken in een groep en hoe ik specifiek kan werken aan een bepaald onderdeel van een project. Ook heb ik geleerd hoe ik mijn vaardigheden op het gebied van programmeren kan toepassen in een praktische setting.

API project

Context en achtergrond: Voor dit project was het mij idee om een API te maken voor een fictieve dierentuin waar alle data van de dierentuin kan opgeslagen worden en dynamisch beschikbaar te is. Daarnaast moesten we ook een andere API consumeren om bij elke nieuwe kat die werd toegevoegd, een leuk feitje over een kat toe te voegen.
Wat heb ik concreet gerealiseerd: Ik heb de API ontworpen en ontwikkeld en heb daarnaast ook de integratie met de andere API geïmplementeerd.
Wat ik heb geleerd: Door dit project heb ik mijn programmeervaardigheden kunnen verbeteren en heb ik geleerd hoe ik een API kan ontwerpen en implementeren. Daarnaast heb ik ook geleerd hoe ik verschillende APIs kan combineren en integreren.

Aquarium automatiseringsproject IOT advanced

Context en achtergrond: Voor dit project was mijn doel om een geautomatiseerd aquarium te creëren dat op een efficiënte manier zijn verlichting regelt, dagelijks voedsel verstrekt en waarbij het waterpeil automatisch wordt aangepast door middel van een pomp. Dit zou zorgen voor een optimale omgeving voor de vissen.
Wat heb ik concreet gerealiseerd: Ik heb het aquarium voorzien van sensoren om de diepte van het water te kunnen meten. Op basis van deze metingen kan de pomp worden ingeschakeld om extra water toe te voegen. Daarnaast heb ik ook een tijdschakelaar geïntegreerd voor de verlichting en voeding. Dit betekent dat de verlichting en voeding op vaste tijden plaatsvinden, waardoor de vissen een consistent ritme hebben.
Wat ik heb geleerd: Door dit project heb ik geleerd hoe ik verschillende sensoren en actuatoren kan combineren om een geautomatiseerd systeem te creëren. Daarnaast heb ik ook geleerd hoe ik de metingen van de sensoren kan gebruiken om beslissingen te nemen en acties uit te voeren. Het was een waardevolle leerervaring voor mij op het gebied van technologie en automatisering.
