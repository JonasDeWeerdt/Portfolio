CyberArk PAM environment

CyberArk is een bedrijf dat zich specialiseert in het beveiligen van gevoelige informatie. Ze doen dit door een Privileged Access Management (PAM) omgeving te creëren waarin alle gevoelige informatie wordt opgeslagen en beveiligd. Deze informatie kan dan enkel worden opgevraagd door de juiste personen. Ik heb een lokaal Windows domein gemaakt waarin ik de CyberArk PAM-omgeving heb geïnstalleerd.

CyberArk Identity Flows

Ik heb de integratie tussen de chatinterface en de CyberArk PAM-omgeving gemaakt met behulp van CyberArk Identity Flows. Dit is een tool waarin no-code automatisaties kunnen gemaakt worden. Ik heb flows gemaakt die acties uitvoeren in de CyberArk PAM-omgeving. Deze flows kunnen dan worden aangeroepen door de chatinterface door gebruik te maken van API's.

Discord bot

Om de integratie mogelijk te maken tussen de chatinterface en de CyberArk PAM-omgeving, heb ik een Discord bot gemaakt. Deze bot kan commando's ontvangen van de gebruiker en deze commando's doorsturen naar Identity Flows. De bot kan ook antwoorden terugsturen naar de gebruiker. De Discord bot maakt API calls naar PAM om de aanpassingen uit te voeren en rechten na te kijken.
